Live API Inspector — Privacy Policy

Last updated: March 2026

What data we collect

Live API Inspector does not collect, store, or transmit any personal data. All network request data captured by the extension exists only in your browser's local memory and is never sent to any external server.

How it works

The extension monitors network requests made by the current browser tab using Chrome's built-in webRequest API. This data is displayed to you in the side panel and is cleared when you close the tab or click the Clear button.

Third parties

We do not share any data with third parties. No analytics, no tracking, no ads.

Permissions

  • webRequest — to observe network requests on the current page
  • tabs — to know which tab is active
  • sidePanel — to display the inspector panel
  • scripting — to enable the request replay feature
